Blogs By Iris - The Developer

Learn Development With Me In Fun Way:

I start by making a brand new project for the tutorial and you'll essentially learn the basics of web development by actually making a new website. This is a great way to remember what you're taught.

Steps To Start From Basic To Advance:

Step 1. Gathering Information: Purpose, Main Goals, and Target Audience ... Step 2. Planning: Sitemap and Wireframe Creation ... Step 3. Design: Page Layouts, Review, and Approval Cycle ... Step 4. Content Writing and Assembly ... Step 5. Coding ... Step 6. Testing, Review, and Launch ... Step 7. Maintenance: Opinion Monitoring and Regular Updating ...

Mistakes That Should be Avoided:

1. Not enabling code quality tools right away ... 2. Using outdated dependencies ... 3. Writing variable names and comments in a language other than English ... 4. Different naming conventions throughout the whole project ... 5. Using meaningless variable names ... 6. Leaving console.logs and to-dos scattered all over the code ... 7. Mixing up async/await, promises and callback syntax ...

Important Points That Makes You Master:

Every educational program helps students develop crucial skills to help with both their personal lives and careers, and a Master in Web Development is no different. Students may gain critical-thinking skills to help solve design problems, and they usually have excellent computer skills.
